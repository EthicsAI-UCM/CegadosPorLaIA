# Código de Conducta de Cegados por la IA

## Nuestra promesa

En Cegados por la IA, queremos crear un ambiente inclusivo, seguro y respetuoso para todos los colaboradores, investigadores, usuarios y la comunidad que interactúa con nuestro proyecto. Nos comprometemos a fomentar la colaboración ética, la diversidad y la accesibilidad.

## Normas generales

Se espera que todos los participantes:

- Traten a los demás con respeto y cortesía.
- Promuevan la inclusión y eviten comentarios discriminatorios o perjudiciales.
- Mantengan la confidencialidad de datos personales o sensibles a los que puedan tener acceso durante el desarrollo o pruebas del sistema.
- Eviten la publicación o difusión de información que pueda comprometer la privacidad de terceras personas.

## Ejemplos de comportamiento inaceptable

- Lenguaje o imágenes ofensivas, insultos, acoso o discriminación.
- Publicar datos personales de usuarios, sujetos de prueba o terceros.
- Usar el proyecto para vigilancia no autorizada o violación de derechos fundamentales.

## Aplicación

Este código de conducta se aplica a todas las interacciones en:

- Repositorio de GitHub
- Canales de comunicación oficiales del proyecto
- Presentaciones, debates y demás eventos asociados al proyecto

## Reconocimiento

Este Código de Conducta se basa en el [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/), adaptado a los objetivos de privacidad y accesibilidad de Cegados por la IA.
